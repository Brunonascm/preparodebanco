🔍 Etapa 1 Ponto de Partida

Antes de iniciar, identifique se o cliente possui conversão de dados de outro sistema.

📦 O cliente tem conversão de dados?

✅ Com conversão
⚠️ Atenção: O preparo de banco deve ser realizado após a conclusão da conversão dos dados.
  • Aguardar a conclusão da conversão antes de iniciar
    Não realizar o preparo com dados incompletos ou em migração.
  • Pegar a planilha da conversão e levantar os regimes com o cliente
    Identificar regime de apuração de cada empresa: Caixa, Competência, Lucro Presumido, etc.
  • Alterar vigência e período inicial nas empresas padrões
    Ajustar conforme a data de início de uso do sistema pelo cliente.
  • Importar os perfis em lote
    Verificar opção "Importar Seleção". Verificar e criar perfil MEI se necessário.
  • Importar RBT12 e Valor Folha
    Utilizar o Importador RBT12 para realizar a importação em lote dos valores.
    📥 Importador RBT12
❌ Sem conversão
  • Pegar a relação de CNPJs do cliente
    Solicitar lista completa de CNPJs para cadastro e configuração.
  • Importar os CNPJs via e-Conversor de Empresas
    Importar os dados base das empresas a partir da relação de CNPJs fornecida pelo cliente.
    🏢 e-Conversor de Empresas

    👥 Listagem eSocial

✅ Etapa 2 Verificação Final

Antes de encerrar o preparo, confirme os itens abaixo.

  • Conferir se todos os regimes tributários estão corretamente configurados
    Simples Nacional (Caixa / Competência / Fator R), Lucro Presumido, etc.
  • Confirmar que os perfis MEI foram criados e configurados
    Verificar DASN-SIMEI e parâmetros específicos do MEI.
  • Validar RBT12 importado com os valores do cliente
    Conferir receita bruta acumulada dos últimos 12 meses para cada empresa.
  • Registrar observações e pendências do cliente
    Documentar qualquer ponto que precise ser resolvido no treinamento.
  • Banco de dados pronto para o treinamento ✅
    Objetivo: chegar ao treinamento já fechando empresas, sem perda de tempo com parametrizações básicas.

🎉 Preparo Concluído!

Todas as etapas foram cumpridas. O banco está pronto para o treinamento.
